Package: org.codehaus.groovy.tools.shell

[Groovy] Class Groovysh

      • org.codehaus.groovy.tools.shell.Groovysh

      • An interactive shell for evaluating Groovy code from the command-line (aka. groovysh). The set of available commands can be modified by placing a file in the classpath named org/codehaus/groovy/tools/shell/commands.xml See XmlCommandRegistrar

        Jason Dillon
