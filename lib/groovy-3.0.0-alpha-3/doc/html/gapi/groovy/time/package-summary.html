Package groovy.time

Package groovy.time Description

Classes for easily manipulating Dates and times. While java.util.Date has GDK methods for adding or subtracting days, this is not so useful for different durations of time. groovy.time.TimeCategory creates a simple internal DSL for manipulating dates and times in a clean and precise fashion.
