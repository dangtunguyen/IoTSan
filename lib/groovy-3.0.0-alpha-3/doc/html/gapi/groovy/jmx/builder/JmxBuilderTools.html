     DynamicMBean.class.isAssignable(cls) ||
     cls.getName().endsWith("MBean") ||
     cls.getName().endsWith("MXBean")
 ) then class is MBean
          Parameters:

          Returns:
          true = if class implements DynamicMBean or interface name that ends in MBean or MXBean.
